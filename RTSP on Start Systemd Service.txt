Notes on Systemd service:
Video: https://www.youtube.com/watch?v=-aKb-k8B8xo 

Change teststream to file name
Go here in terminal: /etc/systemd/system
Sudo nano teststream.service
Do the correct format
Save, exit, check status:
systemctl status teststream.service
Enable: 
sudo systemctl enable teststream.service

Can run with:
sudo systemctl start teststream.service

Stop and disable:
sudo systemctl stop teststream.service
sudo systemctl disable teststream.service
